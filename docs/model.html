Constraints as Differentiable Loss Terms

Standard MSE ignores the physics. VibFormer enforces monotonic health degradation via soft penalty terms — no hard constraints that would block gradient flow.

monotonicity_loss.py
def monotonicity_loss(h: Tensor) -> Tensor:
  # penalise any increase in health index
  deltas = h[:, 1:] - h[:, :-1]
  violations = F.relu(deltas)
  return violations.mean()

def boundary_loss(h_pred, pos):
  # H(0)≈1, H(t_fail)≈0
  early = pos < 0.1
  late  = pos > 0.9
  return (
    F.mse_loss(h_pred[early], ones) +
    F.mse_loss(h_pred[late], zeros)
  )
