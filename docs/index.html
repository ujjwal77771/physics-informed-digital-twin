Physics Constraints
Constraints as Loss Terms
The Physics of Bearing Failure

Bearing degradation is a constrained sequence problem. Standard MSE ignores the physics. VibFormer enforces it via differentiable penalty terms derived from Paris' Law and boundary conditions.

monotonicity_loss.py
def monotonicity_loss(h: Tensor) -> Tensor:
  # penalise any increase in health index
  deltas = h[:, 1:] - h[:, :-1]
  violations = F.relu(deltas)
  return violations.mean()
boundary_loss.py
def boundary_loss(h_pred, pos):
  # H(0)≈1, H(t_fail)≈0
  early = pos < 0.1
  late  = pos > 0.9
  return (
    F.mse_loss(h_pred[early], ones) +
    F.mse_loss(h_pred[late], zeros)
  )
